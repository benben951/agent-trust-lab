# Public Release Checklist

Use this before pushing Agent Trust Lab publicly.

## Safety

- [x] Uses synthetic cases only.
- [x] No real customer data.
- [x] No company policy text.
- [x] No private labels.
- [x] No credentials, tokens, cookies, or API keys.
- [x] Patent disclosure draft is outside the repository.
- [x] Patent claim text is not included.

## Product Evidence

- [x] Static browser demo.
- [x] 10-case synthetic library.
- [x] Batch Markdown trust reports.
- [x] JSON summary output.
- [x] README screenshot.
- [x] Portfolio showcase document.
- [x] Technical report draft.

## Engineering Evidence

- [x] CLI single-case review command.
- [x] CLI batch-review command.
- [x] Unit tests.
- [x] CI workflow.
- [x] GitHub Pages workflow.

## Before Sharing On Resume

- [ ] Confirm GitHub Actions pass.
- [ ] Confirm GitHub Pages URL works.
- [ ] Add repository URL to GitHub profile.
- [ ] Add demo URL to resume project section.
- [ ] Keep patent filing materials private until reviewed.
